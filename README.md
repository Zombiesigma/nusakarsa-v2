🌟 Tentang Nusakarsa
<div style="background: linear-gradient(135deg, #7C6103 0%, #9c7b3c 100%); padding: 1.5rem; border-radius: 15px; color: white; margin: 2rem 0;"> <p style="font-size: 1.2rem; margin: 0;"> Nusakarsa adalah platform serbaguna yang menghubungkan penulis dengan pembaca dalam ekosistem digital yang modern dan imersif. Dibangun dengan teknologi terkini untuk memberikan pengalaman terbaik dalam membaca dan berkarya. Dari puisi hingga novel, semua bersatu dalam satu wadah yang nyaman dan inspiratif. </p> </div>
✨ Fitur Unggulan
📚 Untuk Pembaca
🏷️ Fitur	📝 Deskripsi
🔍 Eksplorasi Karya	Jelajahi ribuan novel dan puisi dengan filter canggih berdasarkan genre, popularitas, atau rekomendasi.
🌓 Mode Membaca	Pilih tema nyaman: Terang, Gelap, Sepia, atau Kertas. Mata tetap segar, imajinasi tetap mengalir.
🎵 Soundtrack	Nikmati musik latar pilihan penulis yang menambah kedalaman saat membaca.
💾 Pustaka Pribadi	Simpan dan kelola koleksi bacaan favoritmu, sync ke semua perangkat.
💬 Komunitas	Beri suka, tulis ulasan, dan diskusi langsung dengan sesama pembaca dan penulis.
✍️ Untuk Penulis
🖋️ Fitur	📝 Deskripsi
🖥️ Studio Modern	Editor Markdown dengan manajemen bab, auto-save, dan pratinjau real-time.
📤 Impor Naskah	Upload file .docx, .pdf, .txt langsung jadi draf. Naskah lama bisa langsung terbit.
🎨 Kustomisasi Karya	Atur sampul, sinopsis, genre, dan visibilitas (publik / draft).
📄 Ekspor PDF Otomatis	Setiap karya yang diterbitkan langsung tersedia dalam format PDF siap unduh.
🛡️ Perlindungan Konten	Anti copy-paste, proteksi unduh gambar, dan watermark opsional untuk menjaga orisinalitas.

🚀 Mulai Mengembangkan di Lokal
📋 Prasyarat
Node.js 18+

npm atau yarn

PostgreSQL (jika ingin menjalankan database lokal)

⚙️ Instalasi
bash
# Clone repositori
git clone https://github.com/zombiesigma/nusakarsa-v2.git

# Masuk ke direktori
cd nusakarsa

# Install dependensi
npm install
# atau
yarn install

# Setup environment (copy .env.example)
cp .env.example .env
# lalu isi variabel yang diperlukan (database, auth, dll)

# Jalankan migrasi database
npx prisma migrate dev

# Jalankan development server
npm run dev
Buka http://localhost:3000 dan lihat hasilnya!
