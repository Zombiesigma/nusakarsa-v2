# **App Name**: Nusakarsa Digital

## Core Features:

- Book Discovery & Browsing: Users can explore a catalog of e-books, view trending titles on the homepage, and search/filter books by various criteria like genre or author.
- Personalized Book Recommendations: An AI tool will suggest e-books to users based on their reading history, preferred genres, and authors, enhancing discovery.
- User Library Management: Users can manage their reading journey by categorizing books into 'Currently Reading', 'Saved' (bookmarks), and 'Finished' lists. This data is persistent.
- Reading Progress & Statistics: Track individual book reading progress, display reading streaks, and other personal statistics on the user's profile.
- Detailed Book View: Provides a dedicated page or modal for each book with detailed information such as synopsis, author, page count, and reader reviews, along with actions like 'Start Reading' or 'Bookmark'.
- Responsive & Animated User Interface: A dynamic and fully animated interface, including particle effects, parallax scrolling in hero sections, smooth page transitions, subtle hover effects, and interactive elements inspired by the provided reference.
- Theme Customization: Allows users to easily toggle between a light and dark theme for improved readability and personalized visual experience.

## Style Guidelines:

- Primary color: A deep, rich burgundy (#91363C) that evokes literary depth and a modern Indonesian cultural elegance, ensuring good contrast on a light background.
- Background color: A subtly warm off-white (#F4EDED) with hints of the primary hue, providing a clean and inviting canvas for content in the light theme.
- Accent color: A vibrant, warm orange (#E6864C) used for calls-to-action, highlights, and interactive elements, designed to complement the primary color and add an energetic touch.
- Headline font: 'Cormorant Garamond' (serif) for an elegant, literary feel. Body text font: 'DM Sans' (sans-serif) for clear readability and a modern aesthetic.
- Utilize clean, line-art style icons, consistent with the reference. Emphasize legibility and modern simplicity.
- Employ a responsive, mobile-first grid system. Features glassmorphism effects for navigation bars, and flexible card layouts that showcase book covers prominently.
- Incorporate a variety of animations: floating particles, parallax scrolling in hero sections, 'fadeUp' and 'float' animations for key elements, ripple effects on buttons, and smooth page transitions as demonstrated in the reference.
